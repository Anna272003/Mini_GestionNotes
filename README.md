# Mini Gestion des Notes – Math & Physique

Ce projet est une **application Java** permettant de gérer les **notes des étudiants** en **Mathématiques** et **Physique**, de calculer les **moyennes**, et de déterminer les **étudiants admis** ou **redoublants**.
Il s’agit d’un **mini-projet académique** réalisé afin de renforcer les bases en **programmation Java**, la manipulation des données et la logique algorithmique.


# Objectifs du projet

- Gérer les informations des étudiants
- Enregistrer les notes en Mathématiques et Physique
- Calculer la moyenne générale
- Identifier les étudiants admis ou redoublants
- Mettre en pratique la logique conditionnelle et les structures de données en Java

# Technologies utilisées

- **Java**
- **IntelliJ IDEA**

# Fonctionnalités principales

- Saisie des informations de l’étudiant
- Saisie des notes :
  - Mathématiques
  - Physique
- Calcul automatique :
  - Moyenne générale

# Règles de calcul (exemple)

- Moyenne générale = (Note Math + Note Physique) / 2
- Étudiant **admis** si moyenne ≥ 10
- Étudiant **redoublant** si moyenne < 10
*(Les règles peuvent être modifiées selon les besoins)
